Algorithm Best Average Worst Space

Merge Sort O(n log n) O(n log n) O(n log n) O(n)

Bubble Sort

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order. It is simple but inefficient for large datasets.

Merge Sort

Merge Sort uses divide-and-conquer. It recursively splits the array and merges sorted halves, making it significantly faster for large inputs.
